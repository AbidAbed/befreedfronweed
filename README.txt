# Be Freed From Weed

Be Freed From Weed is a Web 1 course project done by Aabed Al Aabed and Osama Faza, it is aimed towards educating people about
drugs and their harms.

## Running the Project

The project comes in a ```zip``` file, in it's root directory is a foler named ```Pages``` containing the main page, titled ```index.html```, to run the project one should open this page in their preferred web browser.

## Page Description

The project contains five sub web pages; the [main page] (index.html), the one you are on right now, the
[Information] (Pages/Information.html) page, which contains information about some of the more well known drugs (with more information criteria and
drug types to be added), the [Quiz] (Pages/Game.html) page, which is a fun little game about information found through the site, with much 
more interactivity planned for the JavaScript phase, third we have a form page titled [Contact Us] (Pages/Contact.html), where any viewer can 
submit a contact form with various fields, and last we have the [About Us] (AboutUs.html), which contains a brief description about the 
creators of this page

## Sources of Insperation

During the creation of this project, we gathered insperation from multiple sites these sites and what we gathered from them is listed below,

* [Positive Choice] (https://positivechoices.org.au/.) : We got most of our drug related information from this site.
* [Australian Health Government Website] (https://www.health.gov.au/health-topics/drugs/about-drugs/what-are-drugs) : More drug abuse related information.
* [The Recovery Village] (https://www.therecoveryvillage.com/drug-addiction/types-of-drugs/#gref) : Information about drug groups or families.
* website we couldn't find : insperation for the quiz page of the project.
